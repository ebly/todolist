# TodoList 微信小程序规格文档

## 1. 项目概述

- **项目名称**: Daily Todo
- **项目类型**: 微信小程序
- **核心功能**: 简洁高效的每日待办事项管理
- **目标用户**: 需要日常任务管理的学生/上班族

## 2. UI/UX 规格

### 2.1 页面结构

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | pages/index/index | 今日待办列表 |

### 2.2 布局设计

- **顶部**: 固定标题栏，显示日期和"今日待办"标题
- **主体**: 待办事项列表，支持下拉刷新
- **底部**: 固定输入框，用于添加新待办

### 2.3 视觉设计

**颜色方案**
- 主色: `#1989fa` (Vant 蓝色)
- 背景色: `#f7f8f9` (浅灰白)
- 卡片背景: `#ffffff` (纯白)
- 文字主色: `#323233` (深灰)
- 文字次色: `#969799` (中灰)
- 完成状态: `#c8c9cc` (浅灰)
- 删除色: `#ee0a24` (Vant 红色)

**字体**
- 主字体: 系统默认字体
- 标题: 18px, font-weight: 600
- 正文: 14px
- 次要文字: 12px

**间距**
- 页面边距: 16px
- 列表项间距: 12px
- 内部元素间距: 8px

### 2.4 组件使用

使用 **Vant Weapp** 组件库:
- `van-nav-bar` - 顶部导航栏
- `van-checkbox` - 复选框
- `van-swipe-cell` - 左滑删除
- `van-field` - 输入框
- `van-button` - 按钮
- `van-icon` - 图标
- `van-empty` - 空状态
- `van-dialog` - 对话框
- `van-toast` - 轻提示
- `van-datetime-picker` - 日期选择器(可选)

## 3. 功能规格

### 3.1 核心功能

1. **查看今日待办**
   - 显示当天的所有待办事项
   - 按创建时间倒序排列
   - 显示待办完成状态

2. **添加待办**
   - 点击底部输入框
   - 输入待办内容
   - 点击确认按钮或按回车添加
   - 清空输入框

3. **完成待办**
   - 点击复选框切换完成状态
   - 完成项显示删除线
   - 已完成项排在列表底部

4. **删除待办**
   - 左滑待办项显示删除按钮
   - 点击删除按钮确认删除
   - 使用 van-dialog 二次确认

5. **数据持久化**
   - 使用 wx.setStorageSync 存储到本地
   - 使用 wx.getStorageSync 读取数据

### 3.2 用户交互

- 点击复选框: 切换完成状态
- 左滑列表项: 显示删除按钮
- 点击删除: 弹出确认对话框
- 点击添加按钮: 添加新待办
- 下拉页面: 刷新列表(可选)

### 3.3 边界情况处理

- 输入为空时禁用添加按钮
- 列表为空时显示空状态提示
- 长文本自动换行显示
